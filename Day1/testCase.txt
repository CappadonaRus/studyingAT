Test Case

IDEA: Проверить корректность вывода данных программы в зависимости от передаваемых ей значений, используя для проверки метод вычисления дискриминанты.

Title: Проверка вывода данных программы используя метод вычисления дискриминанты.

Environment (optional):


Pre-condition (optional): 
При дискриминанте равной 0, общие количество корней - 1.
При дискриминанте больше 0, общее количество корней - 2.
При дискриминанте меньше 0, корни мнимые.


Steps:
	
1. Запускаем программу.
	
2. Подаем в метод 3 числа для вычисления дискриминанты. 
	
3. Передаем такие же 3 числа в программу.
	
4. Проверяем правильность вывода данных программы.

	
	
Expected Result:
При дискриминанте = 0, программа должна выводить только 1 действительный корень.
